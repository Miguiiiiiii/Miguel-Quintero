BancoalaMano

DaviPlata es una plataforma de servicios financieros digitales en Colombia que permite a los usuarios realizar transacciones como enviar y recibir dinero, pagar servicios, hacer compras en línea y recargar saldo, todo desde sus teléfonos móviles. Funciona como una billetera digital que se vincula con cuentas bancarias y tarjetas de crédito para facilitar operaciones seguras y convenientes. Utiliza tecnología de banca móvil y sistemas de pago electrónico para garantizar la seguridad de las transacciones y promover la inclusión financiera al hacer accesibles estos servicios a través de dispositivos móviles.

DaviPlata es una línea de negocio de Bancolombia S.A.

Manejo de aplicación

Para ingresar a la aplicación de Bancolombia, se requiere el registro de datos como el tipo de documento, documento y clave de la cuenta.

Al ingresar a la página principal, se muestra el saldo disponible en la cuenta de Bancolombia y la cuenta de ahorros. Además, se encuentran diversas funciones, destacando la opción para enviar dinero en la parte inferior.

En la sección de envío de dinero, se presentan opciones como transferir dinero, usar código QR o utilizar la función TransfiYa entre entidades financieras.

Después de realizar el envío de dinero, ya sea a otros bancos o cuentas de Bancolombia, se genera un comprobante que puede compartirse con otras personas.
